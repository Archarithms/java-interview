Challenge 1
=====================================================
Create a function that takes in a constant-formatted (all caps, words separated by underscores) string and returns the string in “title case”. For example, “THIS_INPUT” should return “This Input”

Step 1 - Clone the project and create a feature branch
Step 2 - Write a Unit Test for the function
Step 3 - Write the function - use all available resources
Step 4 - Test the function
Step 5 - Submit a pull request


Challenge 2
=====================================================
Create a function that takes in a timestamp and returns a String that is the input timestamp formatted as the the month, the day, and the full year. For example, passing a unix epoch time "1499144400” in seconds would return “July 4, 2017”. If no timestamp is given the function should return today’s date. 

Step 1 - Clone the project and create a feature branch
Step 2 - Write a Unit Test for the function
Step 3 - Find and install a library to help
Step 4 - Write the function - use all available resources
Step 5 - Test the function 
Step 6 - pull request
